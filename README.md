### 동작환경

- 개발 환경 : OSX, JAVA 1.8, SpringBoot 2.x, Mysql5.6
- 개발 도구 : STS4, VScode

### 개인용으로 구축하는 방법

1. Clone 또는 Fork 진행
2. JDK 설치 후 STS4 또는 VScode로 진행
3. Maven 프로젝트 업데이트 할 것
4. 이메일 서비스 수정 (현재 AWS로 돌아가는 중)
5. application.properties에서 DataBase 설정
